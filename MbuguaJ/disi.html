What's the Difference?

HTML

Hypertext Markup Language

    Create the structure

  • Controls the layout of the content
  • Provides structure for the web page design
  • The fundamental building block of any web page


CSS

Cascading Style Sheet

    Stylize the website

  • Applise style to the webpage elements
  • Targets variopus screen sizes to make webpages responsive
  • Primarily handles the "look and feel" of a webpage


Javascript

    Increase interactivity

  • Adds interactivity to a webpage
  • Handles complex functions and features
  • Programmatic code whice enhances functionality
